# call-center
este projeto evolui atraves da analise de dados de call center buscando entender e melhorar os processos


Análise Inicial de Dados Simulados de Operações de Call Center															
Objetivo															
															
Realizar uma análise inicial de presença e faltas dos atendentes, utilizando uma base de dados simulada. O propósito é identificar padrões, volume total de faltas e variações por período.															
															
KPIs Desenvolvidos															
															
Total Geral de Faltas															
* Quantidade total de ausências registradas no período.															
															
* Faltas por Mês															
* Distribuição mensal de faltas para identificar tendências ao longo do tempo.															
															
* Faltas por Dia da Semana															
* Análise de padrões de ausência em dias específicos.															
															
															
															
Metodologia															
															
Importação e limpeza da base de dados.															
															
Criação de tabelas dinâmicas para cálculo dos KPIs.															
															
Construção de gráficos e painéis no Google Sheets.															
															
															
Ferramentas Utilizadas															
															
Google Sheets															
															
Tabelas Dinâmicas															
															
Gráficos e Dashboard															
															
Dados simulados para treinamento analítico															
Dashboard															
															
O dashboard foi feito  para a leitura rápida dos KPIs principais, com gráficos . Ele demosntra as informações de faltas por mês, por dia da semana e o total geral.															
															
Insights															
															
Picos de faltas:															
Não foi possível identificar um padrão claro, porque a diferença entre os dias é pequena.															
															
Faltas por mês:															
Também não há uma variação significativa entre os meses.															
															
Expansão da análise:															
A base permite avançar para temas como atrasos, horários reais e produtividade.
